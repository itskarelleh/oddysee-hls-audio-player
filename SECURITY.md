# Security Policy

## Supported Versions

Security updates are provided **only for the latest 0.1.x release**.

| Version | Supported |
| ------- | --------- |
| Latest 0.1.x | ✅ |
| Older versions | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately.

**How to report:**
- Open a GitHub Security Advisory 

Please include:
- A description of the issue
- Steps to reproduce (if applicable)
- Potential impact
- Any relevant logs or screenshots

**What to expect:**
- You should receive an acknowledgment within **72 hours**
- We will investigate and determine severity and scope
- If accepted, we will work on a fix and coordinate disclosure as appropriate

Please **do not publicly disclose** security vulnerabilities until we’ve had a chance to review and address them.
