# HLS Audio Player - Basic Example

A simple demo showing how to use the `@hls-audio-player/core` package.

## Features Demonstrated

- ✅ Basic stream loading and playback
- ✅ Custom header configuration (your killer feature!)
- ✅ Quality level selection
- ✅ Event handling
- ✅ Volume control
- ✅ Real-time status updates

## Getting Started

```bash
cd examples/basic-player
npm install
npm run dev
