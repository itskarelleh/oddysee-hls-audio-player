# Oddysee HLS Audio Player - Basic Example

A simple demo showing how to use the <a href="">oddysee-typescript`</a> package.

Try it out here on  <a href="https://stackblitz.com/github/itskarelleh/oddysee-hls-audio-player/tree/main/examples/basic-player
">StackBlitz</a>

## Features Demonstrated

- ✅ Basic stream loading and playback
- ✅ Custom header configuration (your killer feature!)
- ✅ Quality level selection
- ✅ Event handling
- ✅ Volume control
- ✅ Real-time status updates

## Getting Started

```bash
cd examples/basic-player
npm install
npm run dev
